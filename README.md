# AI Codebase Map Query

This example demonstrates how an AI can efficiently interact with a codebase by first generating a high-level 'map' or summary. Instead of rereading all code files for every query, the AI uses this pre-generated map to quickly identify relevant components, saving time and computational resources. It simulates a codebase, creates a map, and then shows how an AI would use this map to answer queries.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
